2018 Midwest Data Analytics Challenge

Goal:

To predict Minnesota's 2018 voter turnout rates by generating a predictive model



Languages:

R, Python



Tasks:
  1. Obtained FRED and CENSUS data sets using the FRED API (Python)
  2. Cleaned data sets using R's tidyverse and janitor packages (R)
  3. Conducted exploratory data analyses (Python)
  4. Generated a regression tree using R's randomforests package (R)
  5. Used the YouTube API to compare views on Minnesota election-related videos for the years 2010, 2012, 2014, and 2016 to see how strong the correlation between video view count and ultimate voter turnout rate was


Achievements:

Won 7th place out of 50 undergraduate teams
